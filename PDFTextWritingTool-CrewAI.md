အောက်ပါအတိုင်း **Standard Operating Procedure (SOP)** အဖြစ် ရှင်းပြပေးပါမယ်။ ဤလုပ်ငန်းစဉ်သည် **PDFTextWritingTool** ဟုခေါ်သော ကိရိယာကို အသုံးပြု၍ PDF ထဲမှ စာသားများကို အပိုင်းလိုက်ထုတ်ယူခြင်း၊ ဘာသာပြန်ခြင်းနှင့် ဘာသာပြန်ထားသော စာများကို မူရင်းနေရာအတိအကျတွင် ပြန်လည်ရေးသားခြင်းအတွက် ဖြစ်သည်။

## PDF စာသားဘာသာပြန်ပြီး နေရာတကျ ပြန်ရေးနည်း (SOP)

### ၁။ ရည်ရွယ်ချက်
PDF ဖိုင်အတွင်းရှိ စာသားများကို မူရင်းဖွဲ့စည်းပုံမပျက် အပိုင်းလိုက်ဘာသာပြန်ကာ ဘာသာပြန်စာသားများအား သက်ဆိုင်ရာ မူရင်းနေရာများတွင် ပြန်လည်ထည့်သွင်းရန်။

### ၂။ လိုအပ်သောကိရိယာများ
- **PDFTextWritingTool** (စာသားဖတ်ယူခြင်း၊ တည်နေရာကိုဩဒိနိတ် ထုတ်ယူခြင်းနှင့် ပြန်ရေးခြင်းအတွက်)
- ဘာသာပြန်အင်ဂျင် (ဥပမာ - Google Translate API၊ DeepL သို့မဟုတ် လူဖြင့်ဘာသာပြန်ချက်)
- မူရင်း PDF ဖိုင်

### ၃။ လုပ်ဆောင်မှုအဆင့်များ

#### အဆင့် ၁ - စာသားနှင့် တည်နေရာအချက်အလက်များ ထုတ်ယူခြင်း
1. PDFTextWritingTool ကိုဖွင့်ပြီး သင့် PDF ဖိုင်ကို Load လုပ်ပါ။
2. **“Extract Text Blocks”** သို့မဟုတ် **“Extract with Positions”** ဆိုသည့် လုပ်ဆောင်ချက်ကို ရွေးချယ်ပါ။
3. Tool မှ စာသားတစ်ပိုင်းချင်းစီကို အောက်ပါအချက်အလက်များနှင့်အတူ ထုတ်ပေးပါလိမ့်မည်။
   - စာပိုဒ်/စာကြောင်း အညွှန်း
   - မူရင်းစာသား
   - စာသားတည်ရှိသည့် ဘောင်အတိုင်းအတာ (Bounding Box) - စာမျက်နှာနံပါတ်၊ x, y ကိုဩဒိနိတ်၊ အကျယ်၊ အမြင့်
   - ဖောင့်အမည်၊ ဖောင့်အရွယ် (လိုအပ်ပါက)
4. ဤအချက်အလက်များကို ဇယား (CSV/JSON) ပုံစံဖြင့် သိမ်းဆည်းပါ။

#### အဆင့် ၂ - စာသားများကို ဘာသာပြန်ခြင်း
1. ထုတ်ယူထားသော မူရင်းစာသားတစ်ခုချင်းစီကို ဘာသာပြန်ပါ။
   - ကိရိယာတွင် ဘာသာပြန်ချက်ကို အလိုအလျောက် ထည့်သွင်းနိုင်သော API ပါက အသုံးပြုပါ။
   - မပါပါက ပြင်ပဘာသာပြန်ဆော့ဖ်ဝဲဖြင့် ဘာသာပြန်ပြီး ဇယားထဲတွင် **“Translated Text”** ကော်လံတစ်ခု ထပ်ထည့်ပါ။
2. အရေးကြီး - ဘာသာပြန်စာသားကို မူရင်းနေရာတွင် အံဝင်ခွင်ကျဖြစ်စေရန် လိုအပ်လျှင် စာလုံးအရေအတွက်၊ ဖောင့်အရွယ်ကို ချိန်ညှိရန် စဉ်းစားပါ။ Tool က အော်တိုအံဝင်လုပ်ပေးလေ့ရှိသည်။

#### အဆင့် ၃ - ဘာသာပြန်စာများကို PDF ပေါ် ပြန်ရေးခြင်း
1. PDFTextWritingTool တွင် **“Write Translated Text”** သို့မဟုတ် **“Overlay Text”** mode ကိုရွေးပါ။
2. ဘာသာပြန်ချက်များပါဝင်သော ဇယားဖိုင်ကို Import လုပ်ပါ။
3. Tool သည် စာသားတစ်ခုချင်းစီ၏ မူရင်း Bounding Box နေရာကို အသုံးပြုပြီး-
   - မူရင်းစာသားကို ဖျက်ခြင်း သို့မဟုတ် ဖုံးအုပ်ခြင်း (အဖြူရောင် အုပ်ပြီး)
   - ဘာသာပြန်စာသားကို ထိုနေရာတွင် အတိအကျ ရေးသွင်းပေးပါမည်။
4. လိုအပ်ပါက စာမျက်နှာတစ်ခုချင်းစီအတွက် စာသားနေရာချထားမှု၊ ဖောင့်အရောင်၊ နောက်ခံအုပ်ပစ္စည်းတို့ကို Manual ပြင်ဆင်နိုင်သည်။
5. **“Preview”** ဖြင့် ကြည့်ရှုစစ်ဆေးပြီး အဆင်ပြေပါက PDF အသစ်အဖြစ် သိမ်းဆည်းပါ။

### ၄။ စစ်ဆေးအတည်ပြုခြင်း
- ထွက်ရှိလာသော PDF ကိုဖွင့်ပြီး ဘာသာပြန်စာသားအားလုံး မှန်ကန်သောနေရာတွင် ရှိမရှိ၊ စာလုံးပုံစံ ပျက်ယွင်းမှုရှိမရှိ စစ်ဆေးပါ။
- အထူးသဖြင့် ဇယားများ၊ အကွက်များတွင် စာသားပြတ်တောက်မှုမရှိစေရန် သေချာပါစေ။

### ၅။ မှတ်စု
- PDFTextWritingTool သည် Unicode ဖောင့်များကို ထောက်ပံ့မှုရှိရမည်။ မြန်မာစာ ဘာသာပြန်ချက်အတွက် မြန်မာယူနီကုဒ်ဖောင့် ထည့်သွင်းထားရန် လိုအပ်နိုင်သည်။
- မူရင်း PDF ၏ ဖောင့်များ embed လုပ်မထားပါက နေရာအတိအကျ ပြန်ရေးရာတွင် အခက်အခဲရှိနိုင်သဖြင့် Tool ၏ Font Mapping feature ကို အသုံးပြုပါ။
- တရားဝင် စာရွက်စာတမ်းများတွင် မူရင်းအပြင်အဆင် (Layout) ကို ထိန်းသိမ်းရန် ဤနည်းလမ်းသည် အလွန်အသုံးဝင်သည်။

---

ဤ SOP သည် PDFTextWritingTool ၏ လုပ်ဆောင်ချက်များကို အခြေခံထားခြင်းဖြစ်ပြီး သင်အသုံးပြုမည့် ကိရိယာ၏ UI နှင့် လုပ်ဆောင်ချက်အမည်များမှာ အနည်းငယ်ကွဲပြားနိုင်ပါသည်။ လက်တွေ့အသုံးပြုရာတွင် ကိရိယာ၏ အကူအညီစာရွက်စာတမ်း (Help Document) ကို ကိုးကားရန် အကြံပြုပါသည်။
